# German Prepositions Summary

## Prepositions with the Accusative Case

| English Preposition | German Preposition | Example (German)                     | Example (English)                |
| ------------------- | ------------------ | ------------------------------------ | -------------------------------- |
| through             | durch              | Er geht durch den Park.              | He walks through the park.       |
| for                 | für                | Das Geschenk ist für dich.           | The gift is for you.             |
| against             | gegen              | Das Team spielt gegen die Besten.    | The team plays against the best. |
| without             | ohne               | Ich gehe ohne meinen Hund spazieren. | I go for a walk without my dog.  |
| around/about        | um                 | Sie sitzt um den Tisch.              | She sits around the table.       |
|                     |                    | Der Zug kommt um 8 Uhr an.           | The train arrives at 8 o'clock.  |
| until/to            | bis                | Warte bis morgen.                    | Wait until tomorrow.             |
|                     |                    | Der Film dauert bis 22 Uhr.          | The movie lasts until 10 PM.     |
| along               | entlang            | Wir laufen den Fluss entlang.        | We walk along the river.         |

## Prepositions with the Dative Case

| English Preposition  | German Preposition | Example (German)                             | Example (English)                           |
| -------------------- | ------------------ | -------------------------------------------- | ------------------------------------------- |
| out of/from          | aus                | Ich komme aus der Stadt.                     | I come from the city.                       |
|                      |                    | Er trinkt aus der Tasse.                     | He drinks from the cup.                     |
| at/with/by           | bei                | Sie ist bei ihrer Freundin.                  | She is at her friend's place.               |
|                      |                    | Ich arbeite bei Microsoft.                   | I work at Microsoft.                        |
|                      |                    | Beim Essen darf man nicht sprechen.          | One should not talk while eating.           |
| with                 | mit                | Ich gehe mit meinem Bruder ins Kino.         | I go to the cinema with my brother.         |
|                      |                    | Sie schreibt mit einem Stift.                | She writes with a pen.                      |
| after/to             | nach               | Wir fahren nach Berlin.                      | We are driving to Berlin.                   |
|                      |                    | Nach dem Essen machen wir einen Spaziergang. | After eating, we take a walk.               |
| since/for            | seit               | Sie wohnt seit einem Jahr hier.              | She has been living here for a year.        |
|                      |                    | Seit seiner Kindheit mag er Schokolade.      | He has liked chocolate since his childhood. |
| from/of              | von                | Das Geschenk ist von meinem Freund.          | The gift is from my friend.                 |
|                      |                    | Das Buch von Peter liegt auf dem Tisch.      | Peter's book is on the table.               |
| to/at                | zu                 | Wir gehen zu der Schule.                     | We are going to the school.                 |
|                      |                    | Zu Weihnachten besuchen wir unsere Familie.  | At Christmas, we visit our family.          |
| opposite/across from | gegenüber          | Das Hotel liegt gegenüber dem Bahnhof.       | The hotel is opposite the train station.    |
|                      |                    | Sie sitzt mir gegenüber.                     | She sits across from me.                    |

## Prepositions with the Genitive Case

| English Preposition | German Preposition | Example (German)                              | Example (English)                           |
| ------------------- | ------------------ | --------------------------------------------- | ------------------------------------------- |
| during              | während            | Während des Essens sprach niemand.            | During the meal, no one spoke.              |
|                     |                    | Während der Nacht war es ruhig.               | During the night, it was quiet.             |
| despite             | trotz              | Trotz des Regens gingen wir spazieren.        | Despite the rain, we went for a walk.       |
|                     |                    | Trotz seiner Müdigkeit machte er weiter.      | Despite his fatigue, he continued.          |
| because of          | wegen              | Wegen des Staus kamen wir zu spät.            | Because of the traffic jam, we were late.   |
|                     |                    | Wegen der Kinder war es laut.                 | It was noisy because of the children.       |
| instead of          | statt/an Stelle    | Statt des Kuchens nahm sie Obst.              | Instead of the cake, she took fruit.        |
|                     |                    | Anstelle des Autos kaufte er ein Fahrrad.     | Instead of the car, he bought a bike.       |
| within              | innerhalb          | Innerhalb einer Woche war das Projekt fertig. | The project was finished within a week.     |
|                     |                    | Innerhalb des Hauses ist es warm.             | Inside the house, it is warm.               |
| outside of          | außerhalb          | Außerhalb der Stadt ist es ruhiger.           | Outside of the city, it is quieter.         |
|                     |                    | Außerhalb des Geländes ist Parken verboten.   | Parking is prohibited outside the premises. |

## Two-Way Prepositions

| English Preposition | German Preposition | Example (German)                      | Example (English)                     |
| ------------------- | ------------------ | ------------------------------------- | ------------------------------------- |
| at/on               | an                 | Er steht an der Tür.                  | He stands at the door.                |
|                     |                    | Ich denke oft an dich.                | I often think of you.                 |
| on/upon             | auf                | Das Buch liegt auf dem Tisch.         | The book is lying on the table.       |
|                     |                    | Ich freue mich auf die Reise.         | I am looking forward to the trip.     |
| behind              | hinter             | Der Garten ist hinter dem Haus.       | The garden is behind the house.       |
|                     |                    | Er versteckt sich hinter der Tür.     | He is hiding behind the door.         |
| in/into             | in                 | Sie geht in die Küche.                | She goes into the kitchen.            |
|                     |                    | Das Wasser ist in der Flasche.        | The water is in the bottle.           |
| beside/next to      | neben              | Der Hund sitzt neben mir.             | The dog sits next to me.              |
|                     |                    | Neben der Schule ist ein Park.        | Next to the school is a park.         |
| above/over          | über               | Das Flugzeug fliegt über die Stadt.   | The airplane flies over the city.     |
|                     |                    | Wir reden über das Problem.           | We talk about the problem.            |
| under/below         | unter              | Der Ball ist unter dem Tisch.         | The ball is under the table.          |
|                     |                    | Unter den Gästen waren viele Kinder.  | Among the guests were many children.  |
| in front of/before  | vor                | Ich warte vor dem Haus.               | I wait in front of the house.         |
|                     |                    | Vor dem Konzert essen wir etwas.      | Before the concert, we eat something. |
| between             | zwischen           | Der Tisch steht zwischen den Stühlen. | The table is between the chairs.      |
|                     |                    | Zwischen uns herrscht Vertrauen.      | There is trust between us.            |
